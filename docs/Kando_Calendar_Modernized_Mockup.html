What changed vs. the current flat calendar

  • Elevated panel: the whole calendar sits on a surface-1 panel with border + shadow + inset highlight, lifting it off the background — no more flat edge-to-edge grid.
  • Today gets the accent: the current day number is a filled purple circle with glow, instead of a thin outline box.
  • Event chips by priority: due-date items show as colored chips (red/amber/green/purple) with a leading dot — same color language as the board tags, carrying meaning.
  • Softer grid: cell borders use the --border token (not harsh lines); out-of-month days are dimmed; hover highlights the cell.
  • Header polish: month nav arrows are token buttons, "Today" is a bordered pill, the scheduled/unscheduled count uses accent color.
  • Mobile: on a phone this becomes an agenda/list view (upcoming items as a scrollable list of elevated cards) rather than a cramped 7-column grid — a full grid doesn't fit a phone well.
