Animation reminder (the recurring trap)
  • backdrop-filter dies under transform. If you animate the menu open with Framer Motion, put the transform on an outer wrapper and the glass on a static inner div — never on the same element
  • Respect prefers-reduced-motion; animate only transform/opacity; solid-surface fallback where backdrop-filter is unsupported
